# External Integrations

**Analysis Date:** 2026-01-31

## Flake Inputs Integration

**Nixpkgs (nixos-unstable)**
- **Source:** `github:nixos/nixpkgs/nixos-unstable`
- **Purpose:** Primary package source and NixOS modules
- **Usage:** All `pkgs.*` references and system modules
- **Updates:** `nix flake update` or `nix flake lock --update-input nixpkgs`

**Home Manager**
- **Source:** `github:nix-community/home-manager`
- **Input:** `inputs.nixpkgs.follows = "nixpkgs"` (aligned versions)
- **Usage:** User environment configuration via `home-manager.nixosModules.home-manager`
- **Activation:** Both via `nixos-rebuild` (system) and `home-manager switch` (user)

**Niri (sodiboo/niri-flake)**
- **Source:** `github:sodiboo/niri-flake`
- **Input:** `inputs.nixpkgs.follows = "nixpkgs"`
- **Usage:** `programs.niri.enable = true` in `modules/desktop/niri.nix`
- **Package:** `niri` from this flake input

**Noctalia (noctalia-dev/noctalia-shell)**
- **Source:** `github:noctalia-dev/noctalia-shell`
- **Input:** `inputs.nixpkgs.follows = "nixpkgs"`
- **Usage:** 
  - System: `modules/desktop/niri.nix` spawns "noctalia-shell"
  - User: `home/modules/shell/noctalia.nix` imports `inputs.noctalia.homeModules.default`
- **Provides:** Lock screen, session management, status bar widgets
- **Configuration:** Via `programs.noctalia-shell.settings` in home module

## NixOS → Home Manager Integration

**Connection point:** `flake.nix` lines 58-65

```nix
home-manager.nixosModules.home-manager
{
  home-manager.useGlobalPkgs = true;
  home-manager.useUserPackages = true;
  home-manager.extraSpecialArgs = { inherit inputs; };
  home-manager.users.milgraph = import ./home/milgraph.nix;
}
```

**Data flow:**
1. flake.nix passes `inputs` to all modules via `specialArgs`
2. Home Manager module receives `inputs` via `extraSpecialArgs`
3. home/milgraph.nix receives `inputs` in its module arguments
4. Noctalia module imports from `inputs.noctalia.homeModules.default`

**Configuration separation:**
- System packages: `modules/*/*.nix` → `environment.systemPackages`
- User packages: `home/modules/*.nix` → `home.packages`
- Overlap: `home-manager.useGlobalPkgs = true` allows HM to use system pkgs

## Host → Module Integration

**Hardware Configuration:**
- Generated by: `sudo nixos-generate-config --show-hardware-config`
- Location: `hosts/nixos/hardware-configuration.nix`
- Contains: Disk layout, boot modules, CPU microcode
- NOT committed: In `.gitignore`

**Host Configuration:**
- File: `hosts/nixos/configuration.nix`
- Sets: `system.stateVersion`, `nixpkgs.config`, `nix.settings`
- Environment: `environment.sessionVariables` for Wayland/XDG

**Module Imports:**
- Order in flake.nix: Host config first, then modules by category
- Hardware config imported automatically after host config

## Desktop Integration

**Niri Compositor:**
- System packages: `modules/desktop/niri.nix` → `niri`, `xwayland-satellite`
- User config: `home/modules/desktop/niri.nix` (minimal)
- Dotfiles: `home/dotfiles/niri/config.kdl` (keybindings, layout)
- Spawn: `spawn-at-startup "waybar"` in config.kdl

**Wayland Stack:**
- XWayland: `modules/desktop/wayland.nix` → `services.xserver.enable = true`
- XDG Portals: `xdg.portal` with wlr backend for screen sharing
- libinput: `services.libinput.enable = true`

**Noctalia Integration:**
- System: Spawned at Niri startup via config.kdl
- User: Home Manager module with extensive widget config
- Lock screen: Via `include "./noctalia.kdl"` in Niri config
- Status bar: Custom widgets (clock, system monitor, media, etc.)

## Audio Integration

**PipeWire:**
- Module: `modules/desktop/audio.nix`

## Application Integration Patterns

**XDG Desktop Portal:**
```nix
xdg.portal = {
  enable = true;
  wlr.enable = true;
  extraPortals = with pkgs; [ xdg-desktop-portal-gtk ];
};
```

**Enables:**
- Screen sharing in Firefox/Discord
- File pickers in GTK apps
- Screenshot dialogs

**Application dotfiles via XDG:**
```nix
xdg.configFile."alacritty/alacritty.toml".source = ./dotfiles/alacritty/alacritty.toml;
xdg.configFile."alacritty/alacritty.toml".force = true;
```
