  <action>
    Implement run_wizard function in src/wizard.rs:
    
    1. Initialize WizardState
    2. Enter Ratatui terminal:
       - crossterm::terminal::enable_raw_mode()
       - terminal::enable_raw_mode()?
       - Execute ratatui::terminal::Terminal::new()
    
    3. Main loop:
       - Call run_event_loop to get next Event
       - Match on Event:
         - Key(Enter) -> if WelcomeStep, go to next
         - Key(Up/Down) -> navigate within step (if selection)
         - Key(Left) -> go back (if not Welcome)
         - Key(Right) -> go forward (if step complete)
         - Key(Escape) -> go back
         - Key(Tab) -> cycle focus (sidebar vs main area)
         - Char -> pass to current step's handle_input
       - Call current_step.render() for main area
       - Call render_sidebar() for progress
       - frame.render()
    
    4. On exit:
       - Restore terminal: terminal::disable_raw_mode()
       - Clear screen
    
    Update src/main.rs to call run_wizard() instead of placeholder.
